Сведения о программе

Новая ветка развития программы unPacker, объединила доступные инструменты для работы с прошивками Livesuit.
Благодаря стараниям уважаемого Redscorpio, автору imgRePacker.exe возможна более «качественная» запаковка\распаковка прошивок.

Возможности программы

  • Распаковка\запаковка прошивок Livesuit.
  • Распаковка\запаковка графических ресурсов (touchtheme.bin).
  • Конвертирование обоев bmp2bin и bin2bmp.
  • Автоматическая коррекция заголовков у bmp изображений во время запаковки ресурсов (если этого не сделать теряется прозрачность).
  • Создание нулевых файлов для чистки ненужных картинок (нельзя без последствий удалить изображение из файла ресурсов, зато можно заменить файлом размером в 0 байт)

Использование программы

  • Распаковываем прошивку — достаточно перетащить файл с прошивкой в окно программы.
  • Запаковываем прошивку — повторите действия описанные выше, только вместо прошивки перетаскиваем папку «имя_прошивки.img.dump».
  • Распаковываем файл ресурсов touchtheme.bin — перетаскиваем его в окно программы.
  • Запаковываем ресурсы — переместите папку touchtheme.bin.temp в окно программы.
  • Конвертирование обоев — в меню выбираем пункт bin2bmp, затем перетаскиваем файл обоев (например: bg_1.bin) в окно программы на выходе получаем изображение в формате bmp.
    Примечание: конвертация обоев возможна только для устройств с разрешением 800*480 пикселей.
  • Чтобы преобразовать .bmp изображение в файл .bin просто перетащите его в окно программы.
    Формат .bmp изображения должен соответствовать: ширина = 800px, высота = 480px, глубина цвета 24 бита.
  • Замена картинок «пустышками» — в настройках включите команду 0-byte и переместите папку touchtheme.bin.dump в окно программы.

Примечания: Бегущие волны информируют о том, что идёт процесс над упаковкой\запаковкой или другой операцией. По окончанию всех операций, волны станут прямой линией, и появится сообщение о завершении текущих действий.

Меню настройки

  • sc8600 — запаковка прошивки под чип SC8600
  • log — включает логирование, рядом с прошивкой создаётся файл с детальным описанием всех прошедших действий
  • noiso — при распаковке\запаковке iso образы не извлекаются\сохраняются
  • 0-byte — создаёт папку «0_byte» с пустыми файлами, на основе папки «touchtheme.bin.dump»
  • bin2bmp — конвертация обоев bin в bmp

Тесты

Работа программы проверена в следующих ОС: Windows 7 x86\x64, Windows XP x86.
Успешно прошитые устройства: Ritmix RBK450(SC9800), Ainol V6000HDS(SC8600)

  • Добавлена возможность упаковывать прошивку, перетаскиванием папки имя_прошивки.img.dump в окно программы. Выяснилось, что удалять ненужные картинки (которые не используются в прошивке) из файла ресурсов touchtheme.bin нельзя, рушится всё главное меню. Зато можно заменить их пустыми файлами, чтобы уменьшить размер файла ресурсов. Создавать руками тысячу пустых файлов долго, для этого в меню «настройки» есть пункт 0-byte. Когда он активирован, при перетаскивании папки touchtheme.bin.out в окно программы, создаётся папка 0-byte, в которой располагаются пустые файлы, созданные на основе оригинальных файлов. Осталось поместить нужные файлы картинок в эту папку, соглашаясь на замену пустых файлов.
